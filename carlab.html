Project Reports

Project reports for Speed Control and Navigation assignments. Includes technical breakdown, schematics, design decision breakdowns, and lots of data taken from the completed system.

Speed Control: Project goal was to build a system capable of traveling in a straight path at 4 ft/s on flat ground, uphill, and downhill.

Navigation: Project goal was to build a system capable of using visual line following to track a full track twice in a given amount of time.
